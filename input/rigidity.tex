\section{Criteria for rigidity}
Let $k=\mathbb{Q}_p$, and let $L=\mathcal{L}_{n,p}$ be our usual algebra over $\mathbb{Q}_p$.\\
\subsection{The $\mathcal{X}$ criterion}
Let $\mathcal{X}:=\{x\in{L}\setminus\gamma_2L : C_{\sfrac{L}{\gamma_3}}(x)=\mathbb{Q}_px+\gamma_2L\}$, we need to verify that $\mathcal{X}$ generates the entire $L$.\\
Let $x\in{L}\setminus\gamma_2{L}$, that is\\  \[x=a_1e_{1,2}+a_2e_{2,3}+\cdots+a_{n-1}e_{n-1,n}+\sum_{r=2}^{n-1}\sum_{i=1}^{n-r}a_{i,i+r}e_{i,i+r},\] 
where $a_1,a_2,\dots,a_{n-1}$ are not all zero.\\
Let $y\in{C}_{\sfrac{L}{\gamma_3}}(x)$, that is $y=\sum_{r=1}^{n-1}\sum_{i=1}^{i-r}b_{i,i+r}e_{i,i+r}$ such that $[x,y]\in\gamma_3L$.\\
But $[b_{i,i+r}e_{i,i+r},x]\in\gamma_3L$, for every $2\leq{r}\leq{n-1}$ and $1\leq{i}\leq{n-r}$. Thus
\[
[x,y]=\sum_{i=1}^{n-1}\left(a_ib_{i+1}-a_{i+1}b_i\right)[e_{i,i+1},e_{i+1,i+2}]+\gamma_3L=\sum_{i=1}^{n-1}\left(a_ib_{i+1}-a_{i+1}b_i\right)e_{i,i+2}+\gamma_3L
,\]
hence $[x,y]\in\gamma_3L\iff{a}_ib_{i+1}-a_{i+1}b_i=0$, for every $1\leq{i}\leq{n-1}$.\\
But \[a_{i}b_{i+1}-a_{i+1}b_i=0\iff\det\begin{pmatrix}
a_i&a_{i+1}\\
b_i&b_{i+1}
\end{pmatrix}=0\iff\begin{pmatrix}
    b_i&b_{i+1}
\end{pmatrix}=c\begin{pmatrix}
    a_i&a_{i+1}
\end{pmatrix},\]
for some $c\in\mathbb{Q}_p$.\\
Assume $a_i\neq{0}$ for every $1\leq{i}\leq{n-1}$, then 
\begin{align*}
&a_1b_2-a_2b_1=0&\iff&\begin{pmatrix}
    b_1&b_2
\end{pmatrix}=c_1\begin{pmatrix}
    a_1&a_2
\end{pmatrix}.\\
&a_2b_3-a_3b_2=0&\iff&\begin{pmatrix}
    b_2&b_3
\end{pmatrix}=c_2\begin{pmatrix}
    a_2&a_3
\end{pmatrix}.\\
&\vdots&\vdots\\
&a_{n-1}b_n-a_nb_{n-1}=0&\iff&\begin{pmatrix}
    b_{n-1}&b_n
\end{pmatrix}=c_{n-1}\begin{pmatrix}
    a_{n-1}&a_n
\end{pmatrix}.
\end{align*}
But $b_{i+1}=c_ia_{i+1}=c_{i+1}a_{i+1}$ for every $1\leq{i}\leq{n-1}$, hence \\
$c_1=c_2=\cdots=c_{n-1}=c$, which implies that $y=cx+\gamma_2L$ for some $c\in\mathbb{Q}_p$, but this is true for every $c\in\mathbb{Q}_p$, which means that $y=\mathbb{Q}_px+\gamma_2L$ for every $x\in{L}$, but we assumed that $a_i\neq{0}$ for every $1\leq{i}\leq{n-1}$, hence $x\in\mathcal{X}$ has all the elements $e_{i,i+1}$ for $1\leq{i}\leq{n-1}$, thus it generates the entire algebra $L$.\\ 
Assume $a_i=0$ for some $2\leq{i}\leq{n-2}$, and assume that $a_j\neq{0}$ for every $j\neq{i}$, then $a_ib_{i+1}-a_{i+1}b_i=-a_{i+1}b_i=0$ and $a_{i-1}b_i-a_ib_{i-1}=a_{i-1}b_i=0$ if and only if $b_i=0$. But then $\frac{b_1}{a_1}=\frac{b_2}{a_2}=\cdots=\frac{b_{i-1}}{a_{i-1}}=c_1$ and $\frac{b_{i+1}}{a_{i+1}}=\frac{b_{i+2}}{a_{i+2}}=\cdots=\frac{b_{n-1}}{a_{n-1}}=c_2$, for $c_1,c_2\in\mathbb{Q}_p$, hence $y=c_1(a_1e_{1,2}+a_2e_{2,3}+\cdots+a_{n-1}e_{i-1,i})+c_2(a_{i+1}e_{i+1,i+2}+a_{i+2}e_{i+2,i+3}+\cdots+a_{n-1}e_{n-1,n})+\gamma_2L\notin\mathbb{Q}_px+\gamma_2L\implies{x}\notin\mathcal{X}$, which is an example for an element whose centralizers are not necessarily in $\mathbb{Q}_px+\gamma_2L$.
\subsection{The $\mathcal{Y}$ criterion}
For $n=5$.\\
Let $x\in{L}\setminus\gamma_2L$, and assume that $a_i\neq{0}$ for every $1\leq{i}\leq{4}$.\\
Let $y\in{C}_L(x)$, we already know that $b_i=ca_i$, for some $c\in\mathbb{Q}_p$. We look at $(a_{1,2}b_{2,4}-a_{2,4}b_{1,2}+a_{1,3}b_{3,4}-a_{3,4}b_{1,3})e_{1,4}=\\
=(a_{1,2}b_{2,4}-a_{2,4}ca_{1,2}+a_{1,3}ca_{3,4}-a_{3,4}b_{1,3})e_{1,4}=0$.\\
Denote $d=\frac{b_{2,4}}{a_{2,4}}$, then $a_{1,2}(b_{2,4}-ca_{2,4})+a_{3,4}(ca_{1,3}-b_{1,3})=\\a_{1,2}a_{2,4}(d-c)+a_{3,4}(ca_{1,3}-b_{1,3})=0\implies{b}_{1,3}=\frac{a_{1,2}a_{2,4}
(d-c)}{a_{3,4}}+ca_{1,3}$.\\
Also, $(a_{2,3}b_{3,5}-a_{3,5}b_{2,3}+a_{2,4}b_{4,5}-a_{4,5}b_{2,4})e_{2,5}=\\
=(a_{2,3}b_{3,5}-a_{3,5}ca_{2,3}+a_{2,4}ca_{4,5}-a_{4,5}da_{2,4})e_{2,5}=0\implies$\\$\implies{b}_{3,5}=\frac{a_{2,4}a_{4,5}(d-c)}{a_{2,3}}+ca_{3,5}$.\\
Now we look at $(a_{1,2}b_{2,5}-a_{2,5}b_{1,2}+a_{1,3}b_{3,5}-a_{3,5}b_{1,3}+a_{1,4}b_{4,5}-a_{4,5}b_{1,4})e_{1,5}=\\
=(a_{1,2}b_{2,5}-a_{2,5}ca_{1,2}+a_{1,3}(\frac{a_{2,4}a_{4,5}(d-c)}{a_{2,3}}+ca_{3,5})-a_{3,5}(\frac{a_{1,2}a_{2,4}(d-c)}{a_{3,4}}+ca_{1,3})+ca_{1,4}a_{4,5}-a_{4,5}b_{1,4})e_{1,5}=0$.\\
denote $r=\frac{b_{1,4}}{a_{1,4}}$, then $b_{2,5}=\frac{1}{a_{1,2}}(ca_{1,2}a_{2,5}-a_{1,3}a_{3,4}a_{2,4}a_{4,5}(d-c)+\\+a_{1,2}a_{2,4}a_{2,3}a_{3,5}(d-c)-a_{1,4}a_{4,5}(c-r))$.\\
It is readily seen that is $r=d=c$ then
\begin{align*}
&b_{1,2}=ca_{1,2}\\
&b_{2,3}=ca_{2,3}\\
&\vdots\\
&b_{1,4}=ca_{1,4}\\
&b_{2,5}=ca_{2,5},
\end{align*}
which implies that $y\in\mathbb{Q}_px+Z(L)$, but if either $r\neq{c}$ or $d\neq{c}$ then\\ $y$ is not necessarily of this form.\\
Let $z\in{C}_L(y)$. We assumed that $a_i\neq{0}$ for every $1\leq{i}\leq{n-1}$, which implied that $b_i=ca_i$, but $b_iz_{i+1}-b_{i+1}a_i=c(a_iz_{i+1}-a_{i+1}z_i)=0$ hence there exists $\tilde{c}\in\mathbb{Q}_p$ such that $\begin{pmatrix}
    z_1&z_2&\dots&z_{n-1}
\end{pmatrix}=\tilde{c}\begin{pmatrix}
    a_1&a_2&\dots&a_{n-1}
\end{pmatrix}$.\\
Now we look at $(b_{1,2}z_{2,4}-b_{2,4}z_{1,2}+b_{1,3}z_{3,4}-b_{3,4}z_{1,3})e_{1,4}=\\
=(ca_{1,2}z_{2,4}-da_{2,4}\tilde{c}a_{1,2}+(\frac{a_{1,2}a_{2,4}(d-c)}{a_{3,4}}+ca_{1,3})\tilde{c}a_{3,4}-ca_{3,4}z_{1,3})e_{1,4}=0$.
Denote $\tilde{d}=\frac{z_{2,4}}{a_{2,4}}$, then $z_{1,3}=\frac{1}{ca_{3,4}}\left[a_{1,2}a_{2,4}(c\tilde{d}-\tilde{c}d)+(a_{1,2}a_{2,4}(d-c)+ca_{1,3}a_{3,4})\tilde{c}\right]=\frac{1}{ca_{3,4}}\left[a_{1,2}a_{2,4}(c\tilde{d}-\tilde{c}d+\tilde{c}d-c\tilde{c})+c\tilde{c}a_{1,3}a_{3,4}\right]=\frac{1}{ca_{3,4}}\left[a_{1,2}a_{2,4}(c\tilde{d}-c\tilde{c})+c\tilde{c}a_{1,3}a_{3,4}\right]=\frac{1}{ca_{3,4}}\left[ca_{1,2}a_{2,4}(\tilde{d}-\tilde{c})+c\tilde{c}a_{1,3}a_{3,4}\right]=\frac{1}{a_{3,4}}\left[a_{1,2}a_{2,4}(\tilde{d}-\tilde{c})+\tilde{c}a_{1,3}a_{3,4}\right]$, and it is readily seen that if $\tilde{d}=\tilde{c}$ then $z_{1,3}=\tilde{c}a_{1,3}$, but if $\tilde{d}\neq\tilde{c}$ then $z_{1,3}$ is not necessarily of this form, hence $x\notin\mathcal{Y}$ in this case, which shows that an arbitrary $x\in{L}\setminus\gamma_2L$ is not necessarily in $\mathcal{Y}$.
