\section{Criteria for rigidity}
Let $k=\mathbb{Q}_p$, and let $L=\mathcal{L}_{n,p}$ be our usual algebra over $\mathbb{Q}_p$.\\
\subsection{The $\mathcal{X}$ criterion}
Let $\mathcal{X}:=\{x\in{L}\setminus\gamma_2L : C_{\sfrac{L}{\gamma_3}}(x)={k}x+\gamma_2L\}$, we need to verify that $\mathcal{X}=L\setminus\gamma_2L$.\\
Let $x\in{L}\setminus\gamma_2{L}$, that is  $x=a_1e_{1,2}+a_2e_{2,3}+\cdots+a_{n-1}e_{n-1,n}$ where $a_i\neq{0}$.\\
Let $y\in{C}_{\sfrac{L}{\gamma_3}}(x)$, that is $y=b_1e_{1,2}+b_2e_{2,3}+\cdots+b_{n-1}e_{n-1,n}$ where $b_i\neq{0}$, such that $[x,y]\in\gamma_3L$.\\
But
\[
[x,y]=\sum_{i=1}^{n-1}\left(a_ib_{i+1}-a_{i+1}b_i\right)[e_{i,i+1},e_{i+1,i+2}]
,\]
because any other combination $a_ib_j[e_{i,i+1},e_{j,j+k}]$ is either zero or in $\gamma_3L$.\\
Hence, to obtain $[x,y]\in\gamma_3L$ all combinations of the form $a_ib_{i+1}-a_{i+1}b_i$,
for every $1\leq{i}\leq{n}-1$, must vanish. but\\
\[a_{i}b_{i+1}=a_{i+1}b_i\iff\det\begin{pmatrix}
a_i&a_{i+1}\\
b_i&b_{i+1}
\end{pmatrix}=0\iff\begin{pmatrix}
    b_i&b_{i+1}
\end{pmatrix}=c\begin{pmatrix}
    a_i&a_{i+1}
\end{pmatrix},\]
which implies that $y=cx+\gamma_2L$ for some $c\in{k}$, but this is true for every $c\in\mathbb{Q}_p$, which means that $y=kx+\gamma_2L$ for every $x\in{L}\implies\mathcal{X}=L\setminus\gamma_2L$.
