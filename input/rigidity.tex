\section{Criteria for rigidity}
Let $k=\mathbb{Q}_p$, and let $L=\mathcal{L}_{n,p}$ be our usual algebra over $\mathbb{Q}_p$.\\
\subsection{The $\mathcal{X}$ criterion}
Let $\mathcal{X}:=\{x\in{L}\setminus\gamma_2L : C_{\sfrac{L}{\gamma_3}}(x)=\mathbb{Q}_px+\gamma_2L\}$, we need to verify that $\mathcal{X}$ generates the entire $L$.\\
Let $x\in{L}\setminus\gamma_2{L}$, that is\\  \[x=a_1e_{1,2}+a_2e_{2,3}+\cdots+a_{n-1}e_{n-1,n}+\sum_{r=2}^{n-1}\sum_{i=1}^{n-r}a_{i,i+r}e_{i,i+r},\] 
where $a_1,a_2,\dots,a_{n-1}$ are not all zero.\\
Let $y\in{C}_{\sfrac{L}{\gamma_3}}(x)$, that is $y=\sum_{r=1}^{n-1}\sum_{i=1}^{i-r}b_{i,i+r}e_{i,i+r}$ such that $[x,y]\in\gamma_3L$.\\
But $[b_{i,i+r}e_{i,i+r},x]\in\gamma_3L$, for every $2\leq{r}\leq{n-1}$ and $1\leq{i}\leq{n-r}$. Thus
\[
[x,y]=\sum_{i=1}^{n-1}\left(a_ib_{i+1}-a_{i+1}b_i\right)[e_{i,i+1},e_{i+1,i+2}]+\gamma_3L=\sum_{i=1}^{n-1}\left(a_ib_{i+1}-a_{i+1}b_i\right)e_{i,i+2}+\gamma_3L
,\]
hence $[x,y]\in\gamma_3L\iff{a}_ib_{i+1}-a_{i+1}b_i=0$, for every $1\leq{i}\leq{n-1}$.\\
But \[a_{i}b_{i+1}-a_{i+1}b_i=0\iff\det\begin{pmatrix}
a_i&a_{i+1}\\
b_i&b_{i+1}
\end{pmatrix}=0\iff\begin{pmatrix}
    b_i&b_{i+1}
\end{pmatrix}=c\begin{pmatrix}
    a_i&a_{i+1}
\end{pmatrix},\]
for some $c\in\mathbb{Q}_p$.\\
Assume $a_i\neq{0}$ for every $1\leq{i}\leq{n-1}$, then 
\begin{align*}
&a_1b_2-a_2b_1=0&\iff&\begin{pmatrix}
    b_1&b_2
\end{pmatrix}=c_1\begin{pmatrix}
    a_1&a_2
\end{pmatrix}.\\
&a_2b_3-a_3b_2=0&\iff&\begin{pmatrix}
    b_2&b_3
\end{pmatrix}=c_2\begin{pmatrix}
    a_2&a_3
\end{pmatrix}.\\
&\vdots&\vdots\\
&a_{n-1}b_n-a_nb_{n-1}=0&\iff&\begin{pmatrix}
    b_{n-1}&b_n
\end{pmatrix}=c_{n-1}\begin{pmatrix}
    a_{n-1}&a_n
\end{pmatrix}.
\end{align*}
But $b_{i+1}=c_ia_{i+1}=c_{i+1}a_{i+1}$ for every $1\leq{i}\leq{n-1}$, hence \\
$c_1=c_2=\cdots=c_{n-1}=c$, which implies that $y=cx+\gamma_2L$ for some $c\in\mathbb{Q}_p$, but this is true for every $c\in\mathbb{Q}_p$, which means that $y\in\mathbb{Q}_px+\gamma_2L$ for every $x\in{L}$, but we assumed that $a_i\neq{0}$ for every $1\leq{i}\leq{n-1}$, hence $x\in\mathcal{X}$ has all the elements $e_{i,i+1}$ for $1\leq{i}\leq{n-1}$, thus it generates the entire algebra $L$.\\ 
Assume $a_i=0$ for some $2\leq{i}\leq{n-2}$, and assume that $a_j\neq{0}$ for every $j\neq{i}$, then $a_ib_{i+1}-a_{i+1}b_i=-a_{i+1}b_i=0$ and $a_{i-1}b_i-a_ib_{i-1}=a_{i-1}b_i=0$ if and only if $b_i=0$. But then $\frac{b_1}{a_1}=\frac{b_2}{a_2}=\cdots=\frac{b_{i-1}}{a_{i-1}}=c_1$ and $\frac{b_{i+1}}{a_{i+1}}=\frac{b_{i+2}}{a_{i+2}}=\cdots=\frac{b_{n-1}}{a_{n-1}}=c_2$, for $c_1,c_2\in\mathbb{Q}_p$, hence $y=c_1(a_1e_{1,2}+a_2e_{2,3}+\cdots+a_{n-1}e_{i-1,i})+c_2(a_{i+1}e_{i+1,i+2}+a_{i+2}e_{i+2,i+3}+\cdots+a_{n-1}e_{n-1,n})+\gamma_2L\notin\mathbb{Q}_px+\gamma_2L\implies{x}\notin\mathcal{X}$, which is an example for an element whose centralizers are not necessarily in $\mathbb{Q}_px+\gamma_2L$.
\subsection{The $\mathcal{Y}$ criterion}
Let $x=a_ie_{i,i+1}$ for some $1\leq{i}\leq{n-1}$, and let a centralizer element be $y=b_1e_{12}+\cdots+b_{1,n-1}e_{1,n-1}\in{C}_L(x)$, for some $b_1,\dots,b_{1,n-1}\in\mathbb{Q}_p$ not all zero.\\
But \[[x,y]=0\iff\sum_{k=1}^{r-1}a_{j,j+k}b_{j+k,j+r}-a_{j+r,j+k}b_{j,j+k}=0,\]
for all $2\leq{r}\leq{n-1}$ and $1\leq{j}\leq{n}-r$.\\
But every summand of the form $a_{jk}b_{kl}$ vanishes, unless $j=i$ and $k=i+1$, which implies that $a_{i,i+1}b_{i+1,r}=0\iff{b_{i+1,r}=0}$ for all $i+2\leq{r}\leq{n}$, and that $a_{i,i+1}b_{k,i}=0\iff{b_{k,i}=0}$ for all $1\leq{k}\leq{i-1}$.\\
Hence, a generic $y\in{C}_L(x)$ has the form
\[
y=\sum_{r=1}^{n-1}\left[\sum_{j=1}^{i-r-1}b_{j,j+r}+b_{i,i+r}+\sum_{j=i+2}^{n-r}b_{j,j+r}\right],
\]
such that the $b$ coefficients are not all zero.\\
Let $y_{rh}=b_{rh}e_{rh}$ such that $r\neq{i+1}$ and $h\neq{i}$, then $y_{rh}\in{C}_L(x)$.\\
Let $z=d_1e_{12}+\cdots+d_{2,n}e_{2,n}\in{C}_L(y_{rh})$, then all the expressions of the form\\ $d_{kj}b_{jl}$, where $j\neq{r}$ or $l\neq{h}$\\
$d_{jl}b_{kj}$, where $k\neq{r}$ or $j\neq{h}$\\
vanish, because $b_{jl}=b_{kj}=0$.
But all this implies that $d_{kr}b_{rh}=0\iff{d}_{kr}=0$ and $d_{hl}b_{rh}=0\iff{d}_{hl}=0$.\\
